who we are

Laporuns adalah jasa akuntansi untuk UMKM di Indonesia. Kami mengedepankan aspek kemudahan dan ekonomis bagi pengelola UMKM di Indonesia untuk mendapatkan layanan akuntansi secara mudah dan murah.

Dengan Laporuns, kamu bisa fokus melakukan penjualan, memperluas pasar, dan mengembangkan usaha.
